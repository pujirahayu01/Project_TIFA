Bahasa pemrograman


Bahasa pemrograman yaitu sebuah rangkaian kata berbentuk instruksi atau perintah-perintah yang terdiri dari satu atau beberapa baris yang dimengerti oleh komputer. Dalam membuat aplikasi tertentu, digunakan juga bahasa pemrograman yang sesuai dengan kebutuhan aplikasi maka dari itu, seorang developer wajib menguasai atau memahami berbagai bahasa pemrograman tersebut.


Apa itu Bahasa Pemograman ?


Bahasa program adalah sekumpulan instruksi yang diberikan kepada komputer untuk dapat melaksanakan tugas-tugas tertentu dalam menyelesaikan suatu permasalahan. Bahasa program berfungsi untuk memerintah komputer agar dapat mengolah data sesuai dengan langkah - langkah penyelesaian yang telah ditentukan oleh programmer. Bahasa ini memungkinkan seorang programmer untuk menentukan secara persis data mana yang akan diolah oleh komputer, bagaimana data ini akan disimpan/diteruskan, dan jenis langkah apa yang akan diambil dalam berbagai situasi secara persis.

  • python

    python adalah salah satu bahasa yang cukup populer belakangan ini. bahasa ini cukup digemari orang-orang yang ingin berkutik pada data. bahasa ini cocok digunakan oleh pemula karena strukturnya tidak serumit bahasa pemrograman yang lain dan mudah untuk dipahami. Biasanya, bahasa python banyak diterapkan pada pengembangan dan peningkatan IoT, pembuatan software workflow.

  • Java

    Java adalah bahasa pemrograman yang berjalan di berbagai platform, mulai dari komputer pribadi hingga perangkat seluler. Bahasa pemrograman ini awalnya dikembangkan pada tahun 1991 di The Green Project dari Sun Microsystems dan kemudian dirilis oleh Oracle.Nama Java sendiri dipilih karena salah satu pendiri Java, James Gosling, sangat menyukai kopi murni yang digiling langsung di grinder (kopi Tubruk). Ini adalah jenis kopi berasal dari pulau Jawa.

  • C

    Bahasa pemrograman C adalah bahasa pemrograman komputer untuk keperluan umum. bahasa ini ditulis oleh Dennis Ritchie pada tahun 1972 di Bell Telephone Laboratories untuk sistem operasi Unix. Secara desain, fitur C secara jelas mencerminkan kemampuan CPU target. Bahasa C banyak dikembangkan dan digunakan untuk pemrograman sistem komputer jaringan, dan pengembangan aplikasi atau perangkat lunak. Sejak tahun 2000, bahasa C secara konsisten dapat menepati peringkat dua bahasa teratas indeks TIOBE , untuk bahasa pemrograman paling populer.

  • C++

    C++ adalah bahasa pemrograman komputer yang dikembangkan oleh Bjarne Stroustrup dan merupakan evolusi dari bahasa pemrograman C yang dikembangkan oleh Bell Labs (Dennis Ritchie). Bjarne Stroustrup dari Bell Labs pertama kali mengembangkan C++ pada awal 1980-an. Sistem perampingan dan dukungan pemrograman tingkat rendah (pengkodean tingkat rendah) dikembangkan untuk mendukung fitur-fitur C++. Salah satu perbedaan paling mendasar dari bahasa C adalah dukungannya untuk pemrograman berorientasi objek.

  • Kotlin

    Kotlin adalah bahasa pemrograman yang diketik secara statis yang berjalan di Java Virtual Machine atau menggunakan kompiler LLVM yang dapat dikompilasi ke kode sumber JavaScript. Pengembang utamanya adalah tim programmer JetBrains. Kotlin dirancang untuk bekerja sama dengan kode bahasa Java dan mengandalkan kode bahasa Java di perpustakaan kelas Java yang ada seperti Berbagai framework Java yang ada.

  • JavaScript

    Lebih dari 95% situs web yang menggunakan JavaScript membuat namanya sangat populer di kalangan pengembang. Pengembang JavaScript biasanya menggunakannya sebagai ujung depan, berbeda dengan Java dan Python, yang digunakan pengembang sebagai ujung belakang. Framework yang ada pada JavaScript juga banyak digunakan untuk mendukung pengembangan Internet of Things (IoT) pada perangkat mobile.

  • PhP

    PHP adalah bahasa pemrograman yang biasa digunakan untuk pembuatan dan pengembangan web. PHP adalah singkatan dari PHP: Hypertext Preprocessor. Php cocok untuk dipelajari jika kamu ingin menggeluti dunia website. Php adalah dasar yang paling penting untuk mengembangkan sebuah website. Php memiliki struktur yang mudah dan tidak rumit.

  • Perl

    Perl adalah bahasa pemrograman tujuan umum yang awalnya dikembangkan pada mesin Unix oleh Larry Wall. Perl pertama kali dirilis pada 18 Desember 1987, saat Perl 1 dirilis. Dalam versi yang lebih baru, Perl juga tersedia untuk berbagai varian sistem operasi Unix (SunOS, Linux, BSD, HP-UX). Dua fitur utama Perl adalah pengolah kata dan jalan pintas untuk memecahkan masalah umum.

  • VbScript

    VBScript atau Full Visual Basic Scripting Edition adalah bahasa skrip yang diinterpretasikan runtime yang dikembangkan oleh Microsoft Corporation pada tahun 1996, umumnya digunakan di browser web Internet Explorer (untuk Internet Explorer 3.0 dan yang lebih baru). VBScript adalah bahasa scripting yang berasal dari bahasa pemrograman Visual Basic for Applications (VBA) yang digunakan oleh Microsoft Office dan beberapa platform pengembangan Microsoft lainnya. VBScript menghapus beberapa fitur VBA, seperti: B. Kemampuan file I/O dan akses langsung ke sistem operasi menyediakan platform yang aman untuk mengembangkan aplikasi berbasis Web menggunakan platform Active Server Pages (ASP).

  • Html

    Hypertext Markup Language (HTML) adalah bahasa markup standar untuk dokumen yang dirancang untuk dilihat di browser Internet. Teknologi seperti Cascading Style Sheets (CSS) dan bahasa skrip seperti JavaScript dan VBScript dapat membantu di sini. Browser internet menerima dokumen HTML dari server web atau penyimpanan lokal dan mengubah dokumen menjadi halaman web multimedia. HTML menggambarkan struktur semantik halaman web dan isyarat pertama yang terkandung dalam tampilan dokumen. Elemen HTML dijelaskan oleh tag yang diapit tanda kurung siku. Tag /> >.

  • Css

    Cascading Style Sheets (CSS) adalah konvensi untuk mengatur beberapa komponen di web dengan cara yang lebih terstruktur dan konsisten. CSS bukanlah bahasa pemrograman. Demikian pula, gaya dalam pengolah kata seperti Microsoft Word dapat menentukan beberapa gaya, seperti judul, subbab, teks isi, footer, gambar, dan gaya lain yang dibagikan oleh beberapa file. Halaman yang ditampilkan dibuat menggunakan bahasa HTML dan XHTML. CSS mendukung ukuran gambar, warna teks tubuh, warna tabel, ukuran batas, warna batas, warna hyperlink, warna mouseover, spasi antar paragraf, spasi antar teks, kiri, kanan, atas, bawah, dan parameter lainnya dapat dikontrol. CSS adalah bahasa style sheet yang digunakan untuk mendefinisikan tampilan dokumen. CSS memungkinkan Anda untuk merender halaman yang sama dengan cara yang berbeda.

  • Xml

    XML (Extensible Markup Language) adalah bahasa markup tujuan umum yang direkomendasikan oleh W3C untuk membuat dokumen markup untuk pertukaran data antara sistem yang berbeda. XML adalah penerus HTML (HyperText Markup Language), bahasa standar untuk melacak Internet. XML dikembangkan untuk menyimpan data dengan cara yang terdefinisi dengan baik dan dapat dikelola. Kata kunci utama dalam XML adalah data (bentuk jamak dari tanggal), yang dapat memberikan informasi saat diproses. XML menyediakan cara standar yang dapat dimodifikasi untuk menggambarkan konten dokumen. Jadi XML dapat digunakan untuk menggambarkan tampilan database apa pun, tetapi dengan cara yang standar.

  • Framework Css

    • Bootstrap
    Yang pertama adalah Bootstrap. Framework ini memiliki tampilan yang modern, dinamis dan juga user-friendly. Fitur lain dari framework ini adalah website yang dibangun dengan Bootstrap responsif jika dibangun sesuai dengan aturan yang diberikan oleh Bootstrap.
    • Semantic ui
    Kerangka kerja ini memiliki keunggulan bahwa kelas lebih mudah untuk ditulis daripada kerangka kerja lainnya. Selain itu, ia juga menyediakan banyak komponen UI yang berguna untuk membuat HTML Anda terlihat lebih menarik.
    • Materialize
    Kerangka kerja ini dibuat dengan tujuan mengembangkan tema yang memungkinkan pengguna memiliki pengalaman serupa saat menggunakan produk Anda di semua platform. Materialize menggunakan Desain Material, dibuat dan dikembangkan oleh Google.

  • Framework javascript

    • Reactjs
    ReactJS adalah kerangka kerja yang dikembangkan oleh Facebook. Karena ReactJS adalah lintas platform, sering digunakan untuk membuat antarmuka pengguna untuk situs web dan aplikasi seluler. ReactJS adalah inti dari React Native, memungkinkan Anda untuk mengembangkan aplikasi seluler di dua sistem operasi, Android dan iOS, secara bersamaan.
    • Angularjs
    Framework AngularJS ini bersifat open source dan berjalan di sisi client dengan menggunakan konsep MVC untuk membuat tampilan website Anda lebih dinamis.

  • Framework php

    • CodeIgniter
    CodeIgniter, atau singkatnya CI, adalah kerangka kerja dengan arsitektur khas: arsitektur MVC (model, tampilan, pengontrol). Ini memungkinkan Anda untuk menulis kode Anda dengan cara yang lebih terstruktur dan konkret.
    • Laravel
    Laravel biasanya digunakan oleh pengembang untuk mengembangkan situs web yang sangat kompleks. Ini memiliki perpustakaan yang lengkap dan memilih sintaks atau kode yang elegan, ringkas dan jelas.
    • Symfony
    Symfony sangat cocok untuk mengembangkan situs web yang besar dan kompleks. Ini juga mengkonsumsi lebih sedikit memori dan menawarkan kinerja yang lebih tinggi dibandingkan dengan kerangka kerja lainnya.
