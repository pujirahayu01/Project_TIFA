Dasar pemrograman


Dasar Pemograman adalah suatu kegiatan yang sangat penting untuk memperkuat landasan pengetahuan kita mengenai konsep dasar pemograman. Dasar pemrograman yakni sebuah pembelajaran dengan mengajarkan dasar dasar pemrograman secara universal. Apabila kita telah menguasai dasar-dasar pemrograman, kita dapat dengan mudah mempelajari berbagai bahasa pemrograman dengan mudah dan menyenangkan.


Variable & Tipe Data

Variabel

Variable merupakan sebuah wadah untuk menyimpan suatu nilai yang digunakan dalam sebuah progman dan nilai tersebut dapat berubah selama program mengalami proses eksekusi.
contoh : nama, umur, tinggi badan, dll

Konstanta merupakan satuan yang nilainya tidak akan berubah selama program mengalami proses eksekusi. contoh phi dalam sebuah program mencari keliling atau luas lingkaran.

Tipe Data

tipe data merupakan sebuah pengelompokan data sesuai tipe tertentu yang berfungsi untuk memberitahu compiler bagaimana data tersebut ingin di olah.

contoh :
1. String - Untuk kalimat
2. Integer,long, short, byte - untuk bilangan bulat
3. Boolean - untuk data logic
4. Char - untuk data 1 karakter
5. Array - Kumpulan tipe data sejenis


Operator Dasar

Operator adalah suatu simbol yang digunakan untuk memberi perintah kepada komputer yang bergunan untuk melakukan aksi terhadap satu atau lebih operasi.

Operator Aritmatika

Operator Aritmatika dugunakan untuk melakukan operasi Aritmatika yang terdiri dari penjumlahan, pengurangan, perkalian, pembagian dan modulus.

1.Tambahan (+)
Jenis operator aritmatika tambahan dalam bahasa pemrograman dilambangkan dengan simbol "+". Penambahan berguna untuk menambahkan satu operan ke operan lainnya. Misalnya, A = 10 dan B = 5. Jika Anda ingin menambahkan keduanya, gunakan operator '+' di antara keduanya seperti ini: Jika X = A + B, operasinya menjadi X = 15.
2.Pengurangan (-)
Pengurangan jenis operator aritmatika dalam bahasa pemrograman dilambangkan dengan simbol (-). Pengurangan digunakan untuk mengurangi satu operan dari yang lain. Misalnya, X = 21 dan Y = 7. Operasi pengurangan dapat dilakukan dengan menulis N = X - Y dan hasilnya adalah N = 14.
3.Perkalian (*)
Jenis operator perkalian aritmatika dalam bahasa pemrograman dilambangkan dengan simbol (*). Pengurangan berguna untuk mengalikan satu operan dengan operan lainnya. Dalam bahasa pemrograman, perkalian dapat ditulis sebagai a = 5 * 2 atau dengan nilai variabel sebagai a = b * c . Misalnya, jika b = 4 dan c = 5, maka a = 20. Baca juga: Sebelum Anda Menginstal Komputer Anda
4.Distribusi (/)
Jenis operator pembagian aritmatika dalam bahasa pemrograman dilambangkan dengan simbol (/). Pengurangan berguna untuk membagi satu operan dengan operan lainnya. Seperti penjumlahan, pengurangan, dan perkalian, pembagian juga dapat ditulis secara langsung. Misalnya, jika x = 14/2, hasilnya adalah x = 7. Atau tulis dalam variabel seperti x = b / a. Nilai x = 7 jika b = 14 dan a = 2.
5.Modul (%)
Ikon mod adalah contoh operator yang mengembalikan angka dari hasil bagi. Mod atau modulus adalah sisa hasil bagi dalam bahasa pemrograman, ditulis dengan simbol (%). Menurut Khan Academy, operator % biasanya mengembalikan sisanya setelah pembagian untuk menemukan bilangan genap dan ganjil. Misalnya, jika x = 4 % 2, maka x adalah 0. Karena 4 habis dibagi 2. Namun, jika perintahnya adalah x = 5 % 2, maka x adalah 1.


Operator Logika

Operator Logika yaitu operator yang digunakan untuk membuat operasi logika

Mengutip dari buku Sakti Book of PHP Series Web Programming karya Mundzir MF, operator logika digunakan untuk membandingkan dua kondisi logika. Jadi "Benar" berarti logikanya benar, dan "Salah" berarti logikanya salah. Operator logika dasar dari jenis berikut adalah:
1. DAN dan &&
Operator AND dan && mengembalikan nilai logika "True" jika kedua operan benar.
2. OR dan ||
Operator || dan OR mengembalikan true jika salah satu operan benar.
3.XOR
Operator XOR mengembalikan TRUE jika salah satu operan benar tetapi tidak keduanya.
4. NOT
Operator NOT dengan tanda seru (!) mengembalikan nilai true jika salah satu operan salah.
Bug logika sulit dideteksi dengan operasi logika seperti ini, jadi yang terbaik adalah mengujinya terlebih dahulu.
Contoh platform dan situs populer yang menggunakan bahasa pemrograman PHP adalah Facebook, Wikipedia, WordPress, Yahoo!, dan Tumblr.


Operator increment

Increment digunakan untuk menambah nilai variable sebanyak 1 angka. Cara penulisannya menggunakan tanda tambah sebanyak 2 kali untuk Increment
Increment digunakan untuk loop yang nilainya bertambah/bertambah.
Tulis: a++, a+=1, x+=5
2 jenis operator increment :
1. ++$a Pre-increment : menambahkan nilai $a sebanyak 1, lalu kirim nilai.
2. $a++ Post-increment : kirim nilai $a, lalu menambahkan nilai $a sebanyak 1.
Melihat hasilnya, Post-Increment pertama-tama menampilkan hasil dan kemudian menambahkan satu angka ke nilai variabel $a. Namun, tidak seperti Pre-Increment, sejumlah 1 ditambahkan ke $a untuk menampilkan nilai baru.

Operator Relasi

Operator perbandingan/kesetaraan adalah operator yang digunakan untuk menentukan hubungan antara dua operan. Operator perbandingan ditempatkan untuk membandingkan dua ekspresi dan menentukan apakah hasil operasi itu benar. Misalnya, untuk menentukan apakah nilai suatu variabel lebih besar atau sama dengan satu sama lain. Hasil dari operasi tersebut adalah benar (true) atau salah (false).

1. Operator persamaan: Dilambangkan dengan simbol `==`, operator memeriksa apakah dua nilai operan sama. Mengembalikan nilai true jika mereka sama, false sebaliknya. Misalnya, 5==5 benar.
2. Not Equal Operator: Operator yang dilambangkan dengan simbol `!=` memeriksa ketidaksetaraan jika nilai dua operan sama. Mengembalikan nilai true jika tidak sama, false jika tidak. Misalnya, 5!=5 salah.
3. lebih dari operator: lebih-operan, diwakili oleh simbol `>`, menguji apakah operan pertama lebih besar dari operan kedua. Mengembalikan nilai true jika benar, false jika sebaliknya. Misalnya, 6>5 benar.
4. Less Than Operator: Operator kurang dari, diwakili oleh simbol " <", menguji apakah operan pertama lebih kecil dari operan kedua. Mengembalikan nilai true jika benar, false jika sebaliknya. Misalnya: 6<5>=`, operator memeriksa apakah nilai operan pertama lebih besar atau sama dengan nilai operan kedua. Mengembalikan nilai true jika benar, false jika sebaliknya. Misalnya, 5>=5 benar.
6. Kurang dari operator: Operator yang dilambangkan dengan simbol `<=` memeriksa apakah nilai operan pertama lebih kecil atau sama dengan nilai operan kedua. Mengembalikan nilai true jika benar, false jika sebaliknya. Misalnya, 5<=5 benar.


Operator Ternary

Operator Ternary yaitu operator yang sangat unik yang berfungsi untuk membuat seoerti sebuah pertanyaan. Simbol yang digunakan yaitu menggunakan tanda tanya (?) dan titik dua (:) untuk memisahkan jawabannya.

Operator ternary digunakan untuk mempersingkat deskripsi if else, tetapi if else hanya digunakan untuk memberikan nilai pada variabel berdasarkan kondisi tertentu. Sintaks penulisannya adalah:
variabel = kondisi ?if_true:if_false;
return(kondisi?if_true:if_false);
contoh:
#include
using namespace std;
int main(){
int angka;
string ganjil_genap;
cout << "angka:" ; cin>> angka;
ganjil_genap=(angka % 2 ==0)?"genap":"ganjil";
cout << angka << " adalah angka " << ganjil_genap;
return 0;
}
Seperti yang tertulis di atas, operator ternary adalah singkatan dari if yang digunakan untuk memberikan nilai pada suatu variabel. Menggunakan if, kode program di atas menjadi: #include
using namespace std;
int main(){
int angka;
string ganjil_genap;
cout << "angka:" ; cin>> angka;
if(angka % 2 ==0)ganjil_genap="genap";
else ganjil_genap="ganjil";
cout << angka << " adalah angka " << ganjil_genap;
return 0;
}
Pernyataan if lebih panjang, tetapi hasilnya sama dengan contoh sebelumnya. Operator ternary adalah alternatif singkat untuk menggunakan if dan else.
Return dengan Ternary Operator
Terner dapat digunakan untuk mengembalikan nilai dalam suatu fungsi. Fungsi mengembalikan nilai yang tunduk pada kondisi operator ternary.Contoh kode programnya bisa kalian lihat di bawah ini.
#include
using namespace std;
string fungsiku(int angka){
return (angka % 2 ==0)?"genap":"ganjil";
}
int main(){
int angka;
string ganjil_genap;
ganjil_genap=fungsiku(angka);
cout << "angka:" ; cin>> angka;
cout << angka << " adalah angka " << ganjil_genap;
return 0;
}


Operator decrement

Decrement digunakan untuk mengurang nilai variable sebanyak 1 angka. Decrement menggunakan tanda kurang 2 kali dan untuk penempatan kedua tanda tersebut boleh sebelum variable atau sesudah variable.

Decrement digunakan untuk pengulangan dimana nilainya berkurang/berkurang.
Tulis: x--, a-=1, x-=10
2 jenis operator decrement:
1. --$a Pre-decrement : mengurangi nilai $a sebanyak 1, lalu kirim nilai.
2. $a-- Post-increment : kirim nilai $a, lalu mengurangi nilai $a
sebanyak 1.
Melihat hasilnya, Post-Decrement pertama-tama menampilkan hasil dan kemudian menambahkan satu angka ke nilai variabel $a. Namun, tidak seperti Pre-Decrement, sejumlah 1 ditambahkan ke $a untuk menampilkan nilai baru.
